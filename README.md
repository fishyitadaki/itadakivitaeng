# Itadaki Street: Dragon Quest & Final Fantasy 30th Anniversary — English Translation (v1.1)

A fan English translation of いただきストリート ドラゴンクエスト＆ファイナルファンタジー 30th ANNIVERSARY (PS Vita, title ID PCSG00932).

**Status:** The game is fully translated and playable start-to-finish. Version 1.1 brings display and formatting fixes on top of the previous release.

## ⚠️ Requires game update 1.02

The patch is built against the game's **1.02** update. You must have update 1.02 installed first — on an un-updated or mismatched version it won't load correctly and can crash on boot.

## Download & install

Grab the latest build from the [Releases](../../releases) page. Each download includes a README with full step-by-step install instructions for:

- **Real PS Vita** (homebrew + the rePatch reDux0 plugin)
- **Vita3K emulator** (PC / Android)

You must own and have the game installed — the patch only replaces text/UI files, not the game itself.

### DLC overlay (DQ11 add-on owners)

If you own the DQ11 add-on content, you must have the entire contents of the `ACCONVVITADQ11XX` DLC folder in place at `ux0:addcont/PCSG00932/ACCONVVITADQ11XX` **before** installing the patch. Install the patch over the top, overwriting anything it asks to.

## Reporting bugs

Found untranslated text, overflow/cut-off, or odd wording? Open an [issue](../../issues) with a screenshot and where it happens (which screen/menu). Known issues are listed in each release's notes — please check those first.
